Personal Portfolio Website

A responsive, animated personal portfolio website built using HTML, CSS, and JavaScript.
Designed to showcase skills, projects, and contact information with a modern UI and smooth interactions.

Live Preview

(Optional) Add your GitHub Pages / Netlify link here once deployed.

Features

Modern dark UI with animated gradient background

Fixed navigation bar with smooth scrolling

Hero section with animated text and call-to-action

About, Skills, Projects, and Contact sections

Glassmorphism cards with hover effects

Floating particle background animation

Fully responsive layout for desktop and mobile

Clean, readable, and maintainable code structure

Tech Stack

HTML5 – semantic structure

CSS3 – animations, gradients, flexbox, grid, glassmorphism

JavaScript (Vanilla) – smooth scrolling and particle effects

No frameworks. No dependencies.

portfolio/

Sections Overview

Hero – Name, role, short tagline, call-to-action

About – Brief professional summary

Skills – Frontend, backend, and tools

Projects – Highlighted work (currently: MediVault)

Contact – Email, phone, LinkedIn, GitHub

How to Run Locally

Clone the repository

git clone (https://github.com/Siva321-ux/Portfolio)


Open index.html in any modern browser

No build steps required.

Customization

Update personal details in index.html

Modify colors and animations in the <style> section

Add new projects by duplicating a .project-card

Replace placeholder links with real URLs

Future Improvements

Add project images or live demo links

Integrate a contact form with backend/email service

Improve accessibility (ARIA labels, contrast tuning)

Add performance optimizations (reduced animation for low-power devices)
